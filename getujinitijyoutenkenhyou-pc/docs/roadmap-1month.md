# 月次日常点検Webアプリ 1か月計画

## ゴール
- スマホで点検入力（タップで `レ -> ☓ -> ▲ -> 空欄`）
- PCで閲覧・修正
- 管理者印（運行: 岸田 / 整備: 若本）をボタン入力
- 保存・再読込

## 週次計画

### 1週目: MVP入力機能
- 点検表UI
- チェック切替
- 印ボタン
- ローカル保存API

### 2週目: 帳票整形
- 既存Excel様式に合わせた見た目調整
- 1-12月切替
- 運転者/車番別一覧

### 3週目: 管理機能
- 編集履歴（誰がいつ更新したか）
- 検索とフィルタ
- CSV/PDF出力

### 4週目: 本番準備
- バックアップ運用
- 権限管理
- 社内端末テスト（スマホ/PC）
- 利用手順書

## 受け入れ基準
- スマホ片手操作で点検入力できる
- PCで同一データを即時修正できる
- 印の誤入力がない（固定値ボタン）
- 月次データが検索できる
